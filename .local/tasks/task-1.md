# Deep Course Content & Idea Attribution System

## What & Why

The current lesson content stays shallow across all difficulty tiers because the AI prompts give only one-line guidance per level. Users don't feel meaningful progression from Beginner through NextGen, there are no external resource links to take learning further, and the NextGen section has no mechanism for capturing or crediting novel ideas from the community.

This task transforms the platform into a true learning journey:
- Beginner becomes a fun, story-driven hook that makes people want to dive deeper
- Intermediate builds a solid foundation with curated external links (MIT OpenCourseWare, Khan Academy, arXiv, YouTube lectures, open textbooks)
- Advanced covers state-of-the-art research, recent papers, and expert debates
- NextGen opens up to frontier problems, current roadblocks, theoretical directions, community forums, and a lightweight idea attribution system where users can post novel ideas and receive a timestamped Pioneer badge/coin as original creator credit

## Done looks like

- Lesson content at each tier feels substantively different in depth, tone, and focus — Beginner is engaging and accessible, Intermediate teaches mechanisms with real links to go deeper, Advanced covers current research frontiers, NextGen reads like an open research forum
- Each lesson unit displays an "External Resources" section with clickable links (courses, papers, videos, books, forums) appropriate to the difficulty level
- The NextGen tab has an "Idea Board" where users can post original theoretical ideas or proposed research directions
- First poster of an idea receives a visible "Pioneer" badge and a valueless "Nova Coin" credit displayed on their profile and on the idea card
- All idea contributions are timestamped and permanently attributed to the submitting user
- The admin batch-regenerate function uses the new richer prompts
- Existing lessons can be regenerated by admin to pull in the new prompt quality

## Out of scope

- Actual external API calls to verify links (links are AI-suggested and should be reviewed)
- Peer voting or ranking of ideas (future feature)
- Moderation tools beyond basic admin visibility
- Any monetary value attached to Nova Coins

## Tasks

1. **Extend the lesson content JSON schema** — Add `externalResources` (array of `{title, url, type, description}` where type is one of: course, video, paper, book, forum) and `keyTakeaways` (string array) to the lesson content structure. Update TypeScript types used in routes and frontend components.

2. **Rewrite AI content generation prompts** — Replace the thin one-liner difficulty guidelines with detailed, tier-specific instructional briefs. Beginner prompt: engaging story hook, relatable analogies, zero jargon, "why does this matter?" frame, 2-3 beginner-friendly external resources (YouTube explainers, Khan Academy). Intermediate prompt: mechanism explanation, conceptual frameworks, practical case studies, 3-5 links to free online courses and textbooks (MIT OCW, Stanford Online, open textbooks). Advanced prompt: current research landscape, recent landmark papers with arXiv/DOI links, ongoing debates, expert techniques, 3-5 research-grade resources. NextGen prompt: open research questions, known roadblocks and why they're hard, theoretical directions worth exploring, links to active research communities (arXiv categories, Discord servers, Reddit communities, open science forums), invitation to submit an original idea.

3. **Build the Idea Attribution system** — Add a new `ideaContributions` database table (id, userId, topicId, title, description, submittedAt) and a `novaCoins` table (userId, totalCoins, updatedAt). Add API routes: POST to submit an idea (awards 1 Nova Coin to submitter), GET to list ideas for a topic. Display Nova Coin balance on the user profile page alongside XP.

4. **Build the NextGen Idea Board UI** — Add an "Idea Board" panel at the bottom of the NextGen lesson view. Shows existing community ideas as cards with the submitter's display name, Pioneer badge icon, and submission date. Includes a form for submitting a new idea (title + description). After submission, show a celebration moment and explain the Pioneer badge and Nova Coin they earned.

5. **Update the lesson UI to display External Resources** — In the rabbit-hole lesson viewer, add a collapsible "Go Deeper" section below the main lesson content that renders the `externalResources` array as clickable cards with icons by type (book, video, course, paper, forum). Add `keyTakeaways` as a highlighted summary section at the top of each lesson.
